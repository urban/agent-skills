# File Preamble

Use file preambles for discoverability and module-level intent.

## Example: Service file preamble

```typescript
/**
 * @file billing.service.ts
 * @description Billing workflow for invoices, retries, and payment capture.
 * @module billing
 */
```

## Example: Utility module preamble

```typescript
/**
 * @file date-range.ts
 * @description Shared date-range parsing and normalization helpers.
 * @module shared/time
 */
```
